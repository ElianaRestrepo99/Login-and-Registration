# Web Project - Home Page with Login and Registration

This project is a homepage web page that includes a navigation menu, a search area, and a login and registration system. HTML and CSS are used for the structure and visual design, and some icons from the `Boxicons` library are used to improve the user experience.

#Characteristics

# Navigation: Includes links to the "Home", "About", "Services" and "Contact" sections.
# Search bar: Allows users to search within the site.
# Login and Registration: Two forms (login and registration) for users to enter or register.
# Social Networks: Links to social networks (LinkedIn, Instagram, Facebook, Twitter).
# Responsive Design: The design is adapted for an optimal user experience on mobile devices.

##Project Structure

The project is mainly composed of html code.
